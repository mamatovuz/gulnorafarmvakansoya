Gulnora Farm HR Bot

Ishga qabul · Davomat · Oylik · Statistika — bitta Telegram botda

To'liq qo'llanma · Python + aiogram 3.x

1. Bot qaysi muammoni hal qiladi?

Bu dorixona tarmog'i (Gulnora Farm) uchun butun kadrlar (HR) va xodim boshqaruvini bitta Telegram bot ichiga jamlaydi. Ilgari qog'ozda yoki qo'lda qilinadigan ishlar avtomatlashtirilgan.

MuammoBot yechimi
Ishga ariza qog'ozda, tarqoq20 savolli anketa → to'g'ridan-to'g'ri HR ga tushadi
Xodim haqiqatan ishga keldimi?GPS orqali davomat — filial koordinatasidan radius ichida bo'lsagina "keldi" yoziladi
Kim kech qoldi / erta ketdi?Ish vaqtiga qarab avtomatik belgilanadi
Oylik, jarima, to'lovlarBuxgalter panelida yuritiladi
Suhbat vaqtini eslatishBot 1 kun oldin va 2 soat oldin avtomatik eslatma yuboradi
Dam olish kunini almashtirishSo'rov → filial rahbari + HR tasdiqlaydi

Bir jumlada: ishga qabul + davomat + oylik + statistika — hammasi bitta botda.

2. Bot qanday ishlaydi (texnik jihatdan)

  • Til / kutubxona: Python 3.10+ va aiogram 3.x (Telegram Bot API)
  • Baza: SQLite (hrbot.db) — 18 ta jadval (users, applications, attendance, salary_payments va h.k.)
  • Ishga tushishi: bot.py → bazani tayyorlaydi → botni polling rejimida ishga tushiradi
  • Rollarga bo'linish: menyu foydalanuvchining roliga qarab ko'rsatiladi
  • Xavfsizlik: bloklangan foydalanuvchi botdan foydalana olmaydi

8 ta rol: Administrator, HR, Direktor, Buxgalter, Filial rahbari, Farmatsevt, Oddiy xodim, Nomzod (standart).

3. Kim nima qila oladi

🧑 Nomzod (har qanday yangi odam)

  • /start → majburiy obuna yoqilgan bo'lsa, avval kanalga obuna bo'ladi
  • 📝 Ishga ariza topshirish — 20 savolli anketa (ism, tug'ilgan sana, manzil, filial, lavozim, smena, ma'lumot, tajriba, sudlanganlik, oilaviy holat, farzandlar, maosh, Word/Excel, tillar, sabab, telefon, rezyume)
  • Yakunda: ✅ Tasdiqlash · ✏️ Tahrirlash · ❌ Bekor qilish → tasdiqlansa ariza avtomatik HR va Adminlarga yuboriladi
  • 💼 Vakansiyalar ko'radi, 📄 Mening arizalarim holatini kuzatadi
  • Suhbat taklifi kelsa — tasdiqlaydi yoki boshqa vaqt taklif qiladi

🏢 Mavjud xodim (self-registratsiya)

/start → 🏢 Gulnora Farm hodimi tugmasi orqali o'zini ro'yxatdan o'tkazadi (ism, sana, yo'nalish, filial, ish vaqti, oylik, dam kuni, forma, oxirgi 10 kundagi rasm). So'rov HR tasdiqlaydi → xodimga rol va davomat paneli ochiladi.

📍 Tasdiqlangan xodim (davomat)

  • 📍 Ishga keldim — bot GPS so'raydi; filial radiusi (default 150 m) ichida bo'lsa ✅, uzoqda bo'lsa ❌
  • 🏁 Ishdan ketdim — ketish vaqti ham GPS bilan yoziladi
  • Kech qoldi / erta ketti — ish vaqtiga qarab avtomatik
  • 🔄 Dam olish kunini almashtirish — rahbar + HR tasdiqlaydi
  • 👤 Mening profilim — ma'lumot va davomat tarixi

👨‍💼 HR

  • Dashboard — bugungi/haftalik/oylik arizalar va statistika
  • Arizalar — holat bo'yicha filtr, 👁 ko'rish · 📅 suhbat · ✅ qabul · ❌ rad · 📝 izoh
  • Vakansiyalar — yaratish/tahrirlash/yopish/o'chirish
  • Xabarnoma — barchaga / xodimlarga / nomzodlarga / filial bo'yicha (rasm, video, fayl)
  • Qidiruv — ism/telefon/filial/lavozim bo'yicha
  • Xodim self-registratsiyalarini tasdiqlaydi

🧮 Buxgalter

  • Barcha filiallar davomati va kech/erta hisoboti
  • Oylik belgilash/oshirish, oylik berildi/berilmadi, jarima yozish, to'lovlar tarixi
  • Dam olish so'rovlari

👨‍💼 Filial rahbari / 👔 Direktor / 💊 Farmatsevt

  • O'z filiali (yoki barcha filiallar) bo'yicha davomat hisobotlari
  • Filial rahbari — o'z filiali dam olish so'rovlarini tasdiqlaydi

👑 Administrator (bosh boshqaruv)

  • Statistika — foydalanuvchilar, xodimlar, arizalar, eng faol HR
  • Filiallar — qo'shish/tahrirlash/o'chirish + 📍 GPS koordinatasi (davomat uchun shart)
  • Kanallar (majburiy obuna) boshqaruvi
  • Adminlar va HR qo'shish/o'chirish
  • Rollar — istalgan foydalanuvchiga rol berish
  • Audit log — barcha harakatlar sana-vaqti bilan

4. Ishga tushirish qoidalari

python -m venv venv
venv\Scripts\activate            # Windows
pip install -r requirements.txt
copy .env.example .env           # keyin .env ni to'ldiring
python bot.py

.env ichida ikkita narsa shart:

  • BOT_TOKEN — @BotFather'dan olingan token
  • SUPER_ADMINS — sizning Telegram ID (bir nechta bo'lsa vergul bilan). Bu ID lar avtomatik administrator bo'ladi.

5. ⚠️ Muhim qoidalar (aks holda ishlamaydi)

1. Bot faqat BITTA joyda ishlashi kerak.
Ikki joyda (masalan lokal + Railway) bir token bilan ishlatsangiz — har bir xabarga 2 marta javob keladi.
2. Majburiy obuna uchun bot o'sha kanalda administrator bo'lishi shart. Yopiq kanal: -100... ID ko'rinishida; ochiq kanal: @username.
3. Yangi HR/admin qo'shishdan oldin o'sha odam avval botga /start bosgan bo'lishi kerak (ID bazaga tushishi uchun).
4. Davomat ishlashi uchun admin har bir filialga GPS koordinatasini kiritishi shart.
5. .env va hrbot.db ni hech qachon GitHub'ga qo'ymang.
.gitignore allaqachon bularni chiqarib tashlagan — ular ichida token va shaxsiy ma'lumotlar bor.
